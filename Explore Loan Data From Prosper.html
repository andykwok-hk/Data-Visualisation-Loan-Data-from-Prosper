Explore Loan Data From Prosper¶

by Andy Kwok¶

This data set contains 113,937 loans with 81 variables on each loan, including loan amount, borrower rate (or interest rate), current loan status, borrower income, and many others.

In the dataset, we have huge amount of variables and we will focus on a few on them to grasp an overview of the whole dataset. Those variables are Loan Status, Occupation, Employment Status, isborrowerhomeowner and more. We will see how those variables interact with each other through visualising the data.
